# Frontend Assets

This folder contains frontend assets utilities such as font loaders and icon helpers.

- `fonts.ts` — next/font loader for Poppins
- `icons.tsx` — re-exports commonly used `lucide-react` icons

Usage:

- Import `poppins` from `src/frontend/assets/fonts` and include `poppins.variable` in your root `className`.
- Import icons from `src/frontend/assets/icons`.
